Программа предназначена для работы с аудиофайлами игры Star Wars: Knights of the Old Republic I/II.
С её помощью можно извлечь из игры любой аудиофайл, после чего воспроизвести с помощью любого плеера.
Также можно вставить свои аудиофайлы в игру.
Игра использует фальшивые аудиозаголовки, поэтому файлы преобразуются без потери качества.

Для работы программы нужно выбрать папку с файлами и тип преобразования.
В папку помещаются все файлы, которые требуется обработать.
Программа поддерживает расширения mp3 и wav.

- из формата KotOR:
будет создана папка InGeneric, куда будут помещены преобразованные файлы. Настоящее расширение (mp3 или wav) будет определено автоматически.
Внутри неё может быть создана папка Skipped, куда будут помещены файлы, если они были не в формате игры.

- в формат KotOR:
будет создана папка InKotOR, куда будут помещены преобразованные файлы.
Внутри неё может быть создана папка Skipped, куда будут помещены файлы, если файлы изначально были в формате игры.
Расширение файлов будет изменено на wav.

В конце работы программа выдаст сообщение о количестве преобразованных, пропущенных файлов, а также о количестве ошибок, если они произошли в процессе преобразования.
